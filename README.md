# chores-express

Express API server for [Chore Tracker](https://github.com/edwalters99/chores-react)
Please visit the link above for full README file relating to the project.

# Technologies
Project created with:
* Node.js 
* Express
* Mongoose
* MongoDB Atlas

Other dependencies:
* bcryptjs
* colors
* cors
* dotenv
* express-async-handler
* jsonwebtoken
